# Star Trek Explorers

Golang Star Trek game loosely based on the C++ original

## Running Star Trek Explorers

> cd ~/Star-Trek-Explorers
>
> go build
>
> go run ./main.go

## Tests

> cd ~/Star-Trek-Explorers
>
> go test ./...

## Dependencies

Follow the steps for installing golang runtime for your given operating system.

> <https://go.dev/dl/>
